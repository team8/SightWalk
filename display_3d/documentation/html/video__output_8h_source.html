30 // Pangolin video output supports various formats using
31 // different 3rd party libraries. (Only one right now)
32 //
33 // VideoOutput URI's take the following form:
34 // scheme:[param1=value1,param2=value2,...]//device
35 //
36 // scheme = ffmpeg
37 //
38 // ffmpeg - encode to compressed file using ffmpeg
39 // fps : fps to embed in encoded file.
40 // bps : bits per second
41 // unique_filename : append unique suffix if file already exists
42 //
43 // e.g. ffmpeg://output_file.avi
44 // e.g. ffmpeg:[fps=30,bps=1000000,unique_filename]//output_file.avi
