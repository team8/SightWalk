Add series X,Y plot from glsl compatible expressions x_expr, y_expr $i refers to integral index of datum in log.

$0, $1, $2, ... refers to nth series in log. e.g. x_expr = "$i", y_expr = "$0" // index - data[0] plot e.g. x_expr = "$0", y_expr = "$1" // data[0], data[1] X-Y plot e.g. x_exptr ="$i", y_expr = "sqrt($1)} // index - sqrt(data[0]) plot
